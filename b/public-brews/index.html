This website is a collection of all the publicly available brew recipes on the minecraft server Lilac MC. This project was made by vh8t (VH8T) and is open source and can be found on GitHub. If you want to contribute to this project by adding new brews or updating existing ones, you can do so by creating a pull request on the GitHub repository or by contacting me on discord (vh8t). Recipes are from the public lilac recipe list made by Pugs.
